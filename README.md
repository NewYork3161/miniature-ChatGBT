# Mini ChatGBT

![Mini ChatGBT](Images/ChatGBT.png)

## Overview

Mini ChatGBT is an open-source conversational artificial intelligence system designed to demonstrate how modern language models operate internally. The project implements a modular architecture that includes dataset preparation, tokenization, model training, inference, memory handling, vector search retrieval, and conversational interaction.

The system is intentionally structured as a transparent and educational AI framework so developers can inspect and understand each stage of the machine learning pipeline. Every major component is separated into an individual module to encourage experimentation and contributions from the community.

Mini ChatGBT is designed to be both a working conversational AI and a research platform for exploring the mechanics behind language models. The repository exposes the full pipeline used by many modern AI systems including training, tokenization, inference, and contextual memory.

This project is fully open source and welcomes contributions from developers, researchers, and engineers interested in conversational AI.

---

# Project Goals

The primary goals of the project include:

• Demonstrating the architecture behind modern conversational AI systems  
• Providing an educational platform for studying language model mechanics  
• Allowing community contributions and experimentation  
• Enabling reproducible training pipelines  
• Building a modular AI system where each component can be independently modified  

The project emphasizes **clarity, modularity, and extensibility**.
